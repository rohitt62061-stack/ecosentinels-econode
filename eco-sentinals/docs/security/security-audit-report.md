# Security Audit Report - EcoNode Prototyping Phase

## 🛡️ Executive Summary
A comprehensive security review of the **EcoNode** backend (`main.py`) and firmware (`econode.ino`) was conducted applying industry standards (OWASP Top 10).

Current Posture: **High Risk (Lack of Access Controls)**
*Note: Acceptable for isolated local demonstrations, but requires corrections for scale or edge placement.*

## 🔍 Detailed Findings

### 1. Missing Authentication & Authorization
- **Location**: `server/main.py` (All Endpoints)
- **Problem**: Endpoints like `/ingest-sensor` and `/command` do not validate the source identity.
- **Impact**: Any node on the local network can POST false readings or trigger the actuator lid command (`OPEN_LID`).
- **Severity**: 🚨 **Critical**
- **Remediation**: Implement an `API-Key` verification header check behind a FastAPI Depends wrapper.

### 2. Permissive CORS (Cross-Origin Resource Sharing)
- **Location**: `server/main.py` Line 15
- **Problem**: `allow_origins=["*"]` allows any web-page to trigger API payloads directly.
- **Impact**: Potential Cross-Site Request Forgery triggers from browsers facing parallel tabs.
- **Severity**: 🟠 **High**
- **Remediation**: Replace `["*"]` with explicit dashboard targets (e.g., `["http://localhost:5173"]`).

### 3. Resource Exhaustion / Denial of Service
- **Location**: `server/main.py` Line 105 (`/video-feed`)
- **Problem**: Calls `generate_video_frames()` which attempts `cv2.VideoCapture(0)` on every connection trigger.
- **Impact**: Concurrent hits crash on busy hardware drivers or lock out the camera access.
- **Severity**: 🟡 **Medium**
- **Remediation**: Decouple camera indexing into a Singleton thread/reader that streams from a shared buffer array instead of initializing a camera loop in-stream.

### 4. Clear-Text Verification (HTTP)
- **Location**: `firmware/econode.ino` Line 13
- **Problem**: `serverUrl` relies on `http://` streams.
- **Impact**: Wi-Fi packets transparent to packet sniffer frames.
- **Severity**: 🟡 **Medium** (Safe for isolated demo meshes, High for mesh-extender deployments).
- **Remediation**: Use HTTPS (`http.begin("https://...")`) with certificate fingerprint checks where DNS accommodates it properly.
